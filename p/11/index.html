免费苹果开发者帐号也能安装ipa文件

1. 在xcode中配置个人账户

Xcode -> Prefrence -> accounts

怎么配置？谷歌搜 “免费真机调试 iOS”
或者看这里的教程 http://www.cnblogs.com/tandaxia/p/4839997.html

2. 创建一个空应用

File -> New -> Project -> iOS -> Application -> Single View Application -> Next

应用信息随便填

Create 后，点击工程右边的 Fix Issue

3. 下载和使用 iOS App Signer

http://dantheman827.github.io/ios-app-signer/

下载完后打开它， 如果 Provisioning Profile 选项中找不到刚刚创建的应用，重新打开它（应该先 Fix Issue 再打开 iOS App Signer）。

4. 安装重新签名后的ipa文件

在Xcode中 Window -> Devices -> Installed Apps -> '+'
选择重新签名后的ipa文件， 安装到手机后，去设置里面信息自己的证书。

5. 不是所有权限你都可以用

像VPN类应用重新签名也没用，需要付费开发者帐号才可以。
https://developer.apple.com/support/app-capabilities/

此方法来自 http://lifehacker.com/how-to-install-unapproved-apps-on-an-iphone-without-jai-1749519150
